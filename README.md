## Description
This repository contains 5 different Python programs completed as part of the CodeSoft internship.
Each program is console-based, designed for learning and practical application.

---

## Tasks Included

### 1. Daily Study Planner
- **File:** study_planner.py
- **Description:** Manage daily study plans. Users can add plans, view them, and mark them as completed. Plans are saved permanently.
- **Features:** 
  - Add study plans
  - View all plans
  - Mark as completed
  - File handling

### 2. Daily Calculator
- **File:** daily_calculator.py
- **Description:** Perform basic arithmetic operations (Addition, Subtraction, Multiplication, Division) with input validation.
- **Features:**
  - Simple and user-friendly console calculator
  - Handles division by zero

### 3. Daily Quiz Game
- **File:** quiz_game.py
- **Description:** Multiple-choice quiz to test general knowledge. Calculates score and shows correct answers for wrong attempts.
- **Features:**
  - 3 multiple-choice questions
  - Score calculation
  - Correct answer display

### 4. Student Marks Calculator
- **File:** student_marks.py
- **Description:** Calculate total, average, and grade for a student based on marks entered.
- **Features:**
  - Input student name and marks
  - Calculate total and average
  - Assign grade (A/B/C/D)
  - Input validation

### 5. Simple Password Generator
- **File:** password_generator.py
- **Description:** Generates a secure random password with uppercase, lowercase, numbers, and special characters.
- **Features:**
  - User-defined password length
  - Minimum length validation
  - Easy to use console application
